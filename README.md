## 📖 About

PassManager is a full-stack **Password Manager Web Application** built using the **MERN Stack (MongoDB, Express, React, Node.js)**. It allows users to securely store, manage, and access their credentials through a clean and responsive interface.

The application includes a unique feature where users can paste a website link, and it automatically fetches and displays the site's **favicon (icon)**, improving the overall user experience.

This project is available in two versions:
- **Localhost Version** – for testing and local storage
- **MongoDB Version** – for real-world database integration

The goal of this project is to practice and demonstrate full-stack development, API handling, and modern UI design using Tailwind CSS.
